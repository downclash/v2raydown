v7.22.3 · ARM 设备
ARM

适用于 Windows on ARM 设备（如 Surface Pro X、骁龙笔记本等）。同时提供桌面版和控制台版。

下载 ARM64 桌面版 (.zip) .zip 下载 ARM64 便携版 (.zip) .zip

V2RayN (macOS)

v7.22.3 · 跨平台
替代

V2RayN 也提供 macOS 版本，基于 Avalonia 跨平台框架。如果你更习惯 V2RayN 的操作方式，可以在 macOS 上使用。

前往 GitHub Releases 选择 macOS 版

macOS 首次运行 .dmg 时如提示「无法验证开发者」，请前往「系统设置 → 隐私与安全性」点击「仍要打开」。Apple Silicon (M1/M2/M3/M4) 用户请下载 ARM64 版本。

V2RayNG (其他架构)

v2.1.8
更多

为不同 CPU 架构的 Android 设备提供对应版本。不确定架构时请选择上方的 Universal 通用版。
